# Income Prediction Analysis

## Project Overview
This project explores the relationship between socio-demographic factors (Age, Education, Working Hours) and income levels. Using a Logistic Regression model, we achieved an accuracy of **78.34%** in predicting whether an individual earns more than 50K/year.

## Key Features
- Exploratory Data Analysis (EDA) using Seaborn and Matplotlib.
- Data cleaning and preprocessing.
- Statistical modeling with Scikit-Learn.
- Interpretation of socio-demographic indicators.

## Technologies Used
- Python (Pandas, Numpy, Scipy)
- Scikit-Learn (Logistic Regression)
